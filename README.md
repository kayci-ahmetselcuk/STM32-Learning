# STM32 Learning

A comprehensive STM32 learning repository documenting my embedded systems journey with the STM32 NUCLEO-F446RE.

The projects progress from fundamental concepts such as GPIO and interrupts to advanced peripherals including timers, PWM, communication protocols (UART, I2C, SPI), sensors, DMA, and FreeRTOS.

Each project is developed incrementally and focuses on understanding one embedded systems concept at a time.

---

## Hardware

- STM32 NUCLEO-F446RE
- Breadboard
- LEDs
- Push Buttons
- Resistors
- Jumper Wires
- Servo Motor
- OLED Display
- HC-SR04
- DHT22
- MPU6050
